SCANNING AN ARTIFICIAL BRAIN

Neural networks are frequently compared to the human brain so I've taken the metaphor a bit further by making an 'MRI scan' for a popular image classifier, specifically Google's Inception V3. The network is made up of many layers, each containing a multitude of 'neurons' and in this visualiazation, each recangular block is a single layer.

Image data enters the leftmost layer (you can still make out the rough shape of the original image in this layer) and as the signal passes through the network, neurons get excited and light up. The last layer the system guesses what's in the image by highlighting a square. Each square represents one of a thousand categories like 'cat' or 'airplane.'

Use the arrow keys on your keyboard to rotate / zoom in. Created using keras.js and three.js
